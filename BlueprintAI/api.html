2D Primitives

circle(r | d)2d

Creates a 2D circle at the origin. The circle is approximated by a polygon with the number of sides determined by the $fn, $fa, and $fs special variables.

Parameters

r number - Radius of the circle (alternative to d)
d number - Diameter of the circle (alternative to r)

Examples

circle(r = 10);           // Circle with radius 10
circle(d = 20);           // Circle with diameter 20
circle(15);               // Circle with radius 15 (shorthand)
square([width, height], center)2d

Creates a 2D rectangle. If only one dimension is given, it creates a square. The center parameter controls whether the rectangle is centered on the origin.

Parameters

[width, height] vector - Size of the rectangle
center boolean - If true, centers the rectangle on origin (default: false)

Examples

square([20, 10]);          // Rectangle 20x10
square(15);                // Square 15x15
square([30, 20], center=true); // Centered rectangle
polygon(points, paths, convexity)2d

Creates a 2D polygon from a list of points. Points are specified as [x,y] coordinates. Multiple paths can be used to create polygons with holes.

Parameters

points vector of vectors - List of [x,y] coordinates
paths vector of vectors - Optional path indices (default: all points in order)
convexity integer - Maximum number of ray intersections (default: 1)

Examples

polygon(points=[[0,0],[10,0],[10,10],[0,10]]);
polygon(points=[[0,0],[20,0],[10,20]], paths=[[0,1,2]]);
polygon(points=[[0,0],[10,0],[10,10],[0,10],[3,3],[7,3],[7,7],[3,7]],
        paths=[[0,1,2,3],[4,5,6,7]]); // Polygon with hole

3D Primitives

cube([width, height, depth], center)3d

Creates a 3D rectangular prism (cube). If a single number is given, it creates a perfect cube. The center parameter controls positioning relative to the origin.

Parameters

[width, height, depth] vector - Dimensions of the cube
center boolean - If true, centers the cube on origin (default: false)

Examples

cube([20, 10, 5]);         // Rectangular prism
cube(15);                  // Cube 15x15x15
cube([10, 10, 10], center=true); // Centered cube
sphere(r | d)3d

Creates a 3D sphere at the origin. The sphere is approximated by a polyhedron with resolution controlled by the $fn, $fa, and $fs special variables.

Parameters

r number - Radius of the sphere (alternative to d)
d number - Diameter of the sphere (alternative to r)

Examples

sphere(r = 10);            // Sphere with radius 10
sphere(d = 20);            // Sphere with diameter 20
sphere(15);                // Sphere with radius 15
cylinder(h, r1 | d1, r2 | d2, center)3d

Creates a 3D cylinder (or cone/frustum). If r2/d2 is specified and different from r1/d1, it creates a conical shape. The center parameter controls vertical positioning.

Parameters

h number - Height of the cylinder
r1 | d1 number - Bottom radius/diameter
r2 | d2 number - Top radius/diameter (optional, defaults to r1/d1)
center boolean - If true, centers vertically on origin (default: false)

Examples

cylinder(h=20, r=5);       // Cylinder
cylinder(h=15, r1=8, r2=3); // Cone
cylinder(h=10, d=12, center=true); // Centered cylinder
polyhedron(points, faces, convexity)3d

Creates a 3D polyhedron from a list of points and triangular faces. This is the most general 3D primitive, allowing creation of any polyhedral shape.

Parameters

points vector of vectors - List of [x,y,z] coordinates
faces vector of vectors - List of triangular face indices
convexity integer - Maximum number of ray intersections (default: 1)

Examples

polyhedron(
  points=[[10,10,0],[10,-10,0],[-10,-10,0],[-10,10,0],[0,0,10]],
  faces=[[0,1,4],[1,2,4],[2,3,4],[3,0,4],[1,0,3],[2,1,3]]
);

Transformations

translate([x, y, z])transform

Moves child objects by the specified vector. Translation is applied to all children within the module.

Parameters

[x, y, z] vector - Translation vector

Examples

translate([10, 0, 0]) cube(5);     // Move cube 10 units right
translate([0, 5, 0]) sphere(3);     // Move sphere 5 units forward
translate([2, 3, 4]) {              // Move multiple objects
  cube(5);
  cylinder(h=10, r=2);
}
rotate([x, y, z] | a, [x, y, z])transform

Rotates child objects around the origin. Can specify individual axis rotations or a single angle around a custom axis. Angles are in degrees.

Parameters

[x, y, z] vector - Rotation angles around X, Y, Z axes
a, [x, y, z] number, vector - Single angle and axis vector

Examples

rotate([0, 0, 45]) square(10);     // Rotate 45° around Z
rotate([30, 0, 0]) cube(5);         // Rotate 30° around X
rotate(a=45, v=[1,1,0]) cylinder(h=10, r=2); // Rotate around custom axis
scale([x, y, z])transform

Scales child objects by the specified factors. Values greater than 1 enlarge, less than 1 shrink. Negative values mirror across that axis.

Parameters

[x, y, z] vector - Scale factors for each axis

Examples

scale([2, 1, 1]) cube(5);          // Double width
scale([0.5, 0.5, 0.5]) sphere(10);  // Half size
scale([1, -1, 1]) cube(5);          // Mirror across Y axis
mirror([x, y, z])transform

Mirrors child objects across a plane perpendicular to the specified normal vector. The vector doesn't need to be normalized.

Parameters

[x, y, z] vector - Normal vector of the mirror plane

Examples

mirror([1, 0, 0]) cube(5);         // Mirror across YZ plane
mirror([0, 1, 0]) sphere(5);        // Mirror across XZ plane
mirror([1, 1, 0]) cylinder(h=10, r=2); // Mirror across diagonal plane

Boolean Operations

union()boolean

Combines all child objects into a single solid. This is the default operation when multiple objects are rendered together.

Parameters

No parameters - operates on all children

Examples

union() {
  cube([10, 10, 10]);
  translate([5, 5, 5]) sphere(7);
}

cube(10);
translate([5, 5, 5]) sphere(7); // Same as above (implicit union)
difference()boolean

Subtracts all subsequent child objects from the first child object. The first child defines the base shape, and all others are cut out of it.

Parameters

No parameters - operates on all children

Examples

difference() {
  cube([20, 20, 20]);           // Base shape
  translate([5, 5, 5]) sphere(8); // Hole
}

difference() {
  cylinder(h=10, r=5);
  translate([0, 0, 5]) cylinder(h=6, r=3);
}
intersection()boolean

Creates a shape that contains only the volume where all child objects overlap. The result is the common intersection of all children.

Parameters

No parameters - operates on all children

Examples

intersection() {
  cube([20, 20, 20]);
  translate([10, 10, 10]) sphere(15);
}

intersection() {
  cylinder(h=20, r=8);
  cube([15, 15, 20], center=true);
}

Mathematical Functions

abs(x)math

Returns the absolute (positive) value of a number. Negative numbers become positive, positive numbers remain unchanged.

Parameters

x number - Input value

Examples

abs(-5);     // Returns 5
abs(3.14);   // Returns 3.14
abs(-0);     // Returns 0
sin(x), cos(x), tan(x)math

Trigonometric functions. Note: OpenSCAD uses degrees, not radians, unlike many programming languages.

Parameters

x number - Angle in degrees

Examples

sin(30);     // Returns 0.5
cos(60);     // Returns 0.5
tan(45);     // Returns 1

// Parametric circle
for(angle = [0:10:360]) {
  translate([cos(angle)*10, sin(angle)*10, 0])
    sphere(1);
}
sqrt(x)math

Returns the square root of a number. Returns undef for negative numbers.

Parameters

x number - Input value (must be non-negative)

Examples

sqrt(4);     // Returns 2
sqrt(2);     // Returns 1.41421...
sqrt(0);     // Returns 0
min(x, y, ...), max(x, y, ...)math

Returns the minimum or maximum value from a list of numbers. Can take multiple arguments.

Parameters

x, y, ... numbers - Values to compare

Examples

min(5, 3, 8, 1);    // Returns 1
max(5, 3, 8, 1);    // Returns 8
min([2, 7, 1, 9]);  // Returns 1 (works with vectors too)
rands(min, max, count, seed)math

Generates a list of random numbers within a specified range. The seed parameter ensures reproducible results.

Parameters

min number - Minimum value (inclusive)
max number - Maximum value (exclusive)
count integer - Number of random values to generate
seed number - Optional seed for reproducible results

Examples

rands(0, 10, 5);        // 5 random numbers 0-10
rands(0, 1, 3, 42);     // Reproducible random numbers

// Random spheres
for(pos = rands(-10, 10, 6)) {
  translate([pos[0], pos[1], pos[2]])
    sphere(1);
}

Special Variables & Constants

$fn, $fa, $fsspecial

Control the resolution/quality of curved surfaces. $fn sets minimum number of fragments, $fa sets minimum angle, $fs sets minimum size. Higher values = smoother curves but slower rendering.

Variables

$fn integer - Minimum number of fragments (default: 0, uses $fa/$fs)
$fa number - Minimum angle in degrees (default: 12)
$fs number - Minimum fragment size (default: 2)

Examples

$fn = 50; sphere(10);     // High quality sphere
$fa = 5; $fs = 0.5; cylinder(h=20, r=5); // Fine resolution
$fn = 6; cylinder(h=10, r=5); // Hexagonal prism
PIconstant

Mathematical constant π (pi), approximately 3.141592653589793. Used in trigonometric calculations and circle geometry.

Examples

circumference = 2 * PI * radius;
area = PI * pow(radius, 2);

// Parametric equations
x = cos(angle) * radius;
y = sin(angle) * radius;
$children, $tspecial

Special variables available within modules. $children contains the number of child objects. $t is the animation time parameter (0-1).

Variables

$children integer - Number of child objects passed to the module
$t number - Animation time (0.0 to 1.0)

Examples

module repeat() {
  for(i = [0 : $children-1]) {
    translate([i*20, 0, 0]) children(i);
  }
}

repeat() {
  sphere(5);
  cube(5);
}

// Animation
rotate([0, 0, $t * 360]) cube(10);
