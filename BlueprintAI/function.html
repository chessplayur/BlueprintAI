OpenSCAD Guide: Functions & Modules

A detailed breakdown of the OpenSCAD Cheatsheet entries.

2D Primitives

circle(r | d)

Creates a circle at the origin. Use r for radius or d for diameter. Result is a 2D surface.

square([w, h], center)

Creates a rectangle with width w and height h. If center=true, it centers on the origin.

polygon([points], [paths])

Creates a multi-sided shape by connecting a list of [x,y] coordinates.

3D Primitives

sphere(r | d)

Creates a 3D sphere at the origin. Resolution is controlled by the $fn variable.

cube([x, y, z], center)

Creates a 3D rectangular prism. If a single value is given (e.g., cube(10)), it makes a perfect cube.

cylinder(h, r|d, center)

Creates a cylinder. You can use r1 and r2 to create cones or frustrated cones.

Transformations

translate([x, y, z])

Moves child objects along the X, Y, and Z axes by the specified distances.

rotate([a, b, c])

Rotates child objects by angles a, b, c (in degrees) around the X, Y, and Z axes respectively.

scale([x, y, z])

Resizes child objects by a multiplier. [2, 2, 2] doubles the size in all directions.

mirror([x, y, z])

Mirrors the object across a plane perpendicular to the specified vector.

Boolean Operations

union()

Combines all child objects into a single solid. This is the default behavior in OpenSCAD.

difference()

Subtracts the second (and subsequent) child objects from the first child object.

intersection()

Keeps only the portion of space where all child objects overlap.

Mathematical Functions

abs(x)

Returns the absolute (positive) value of a number.

sin(x) / cos(x) / tan(x)

Standard trigonometric functions. Note: OpenSCAD uses degrees, not radians.

sqrt(x)

Returns the square root of a number.

rands(min, max, count)

Generates a list of random numbers within a specific range.
